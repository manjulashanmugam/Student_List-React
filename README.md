# Student List React App

A simple **React** application to manage and display a list of students.  
This project demonstrates basic React concepts like state management, components, and CRUD operations.

## Features

- Display a list of students
- Add new students
- Delete students
- Simple and responsive UI

## Tech Stack

- React.js
- CSS
- npm
